Overview

케이크/타르트를 전문으로 판매하는 프리미엄 디저트 카페 브랜드 '오디데이(OH!D-DAY)'입니다.
- 시선을 끄는 큰 사이즈의 메인 배너가 상단에 위치하고 바로 하단에는 인기 메뉴를 배치하여 사용자가 쉽게 이용할 수 있도록 하였습니다.
- 인기 메뉴 아래로 브랜드의 대표 서비스인 정기 배송 주문과 픽업 예약 서비스를 배치하고, 진행 중인 이벤트들이 한 개씩 슬라이드 되도록 구현하였습니다.
- 사이트 하단에는 브랜드의 sns 계정에 업로드 된 이미지들을 흐르는 배너로 보여주고, 이미지 클릭 시 새 창으로 sns 계정 페이지를 확인할 수 있습니다.

Brand Info

About

Brand Story

자주 먹는 디저트가 우리집 앞까지!
'식재료, 샐러드, 도시락처럼 디저트도 매번 직접 구매하러 가지 않고 집에서 편하게 정기 배송을 받을 수 없을까?'
- 디저트를 좋아하는 사람들이 디저트를 더 편하게 즐길 수 있는 '오디데이(OH!D-DAY)' 브랜드를 기획하였습니다.
- '오디데이(OH!D-DAY)' 웹사이트 사용자는 선택한 디저트를 정기적으로 배송 받을 수 있고, 예약한 디저트를 오프라인 매장에서 픽업할 수 있습니다.

Logo Story

- '오디데이(OH!D-DAY)'는 정기배송 서비스를 제공하는 디저트 카페로, '오늘은 디저트를 먹는 날'이라는 의미를 가진 브랜드명입니다.
- 'D-DAY'에 '오늘이 디저트를 먹는 당일'이라는 의미가 있고, '오'늘은 '디'저트를 먹는 날의 준말이라는 의미도 담아 '오디데이'를 네이밍하였습니다.
- 'D-DAY'의 'D-D'를 눈으로 표현하고 눈썹과 입 모양을 그려 디저트를 먹는 귀여운 표정을 로고에 담았습니다.
- 기쁜 마음을 표현하기 위해 느낌표를 넣고 부드러운 느낌의 필기체 폰트를 사용하여 브랜드 로고를 제작하였습니다.

Layout Main Page

Main Banner

넉넉한 사이즈로 웹사이트에 방문하였을 때 디저트 이미지가 시선을 사로잡도록 하였습니다. 첫번째 배너는 디저트 이미지와 함께 브랜드명을 표기하였고, 이어지는 배너들은 신메뉴과 인기메뉴를 보여주어 사용자의 흥미를 끌도록 하였습니다.

Best Menu

브랜드를 대표하는 인기메뉴를 사용자가 쉽고 빠르게 확인할 수 있도록 페이지의 상단에 배치했습니다. 원형을 사용함으로써 부드러운 느낌을 주었고, 두번째 페이지의 마지막 원형은 더보기 버튼으로 활용하였습니다. 화살표 클릭 시 한 페이지당 네 메뉴씩 페이지가 넘어갑니다.

Order

브랜드의 대표 서비스인 정기 배송 신청과 매장 픽업 예약 페이지로 이동할 수 있는 영역입니다. 각 서비스에 대한 간단한 설명 문구와 함께 이미지를 교차하여 배치했습니다, 스크롤 시 아래에서 위로 나타나는 애니메이션 효과를 주었습니다.

Store

오프라인 매장 검색과 가맹점 개설 문의 서비스 버튼을 큰 사이즈로 배치해 매장 검색과 가맹점 개설 문의를 쉽게 할 수 있도록 하였습니다.

Event

진행 중인 이벤트나 신메뉴 출시 등의 소식을 보여줍니다. 자동으로 한 개씩 슬라이드 되며, 하단의 화살표 버튼으로도 넘겨볼 수 있습니다. 모바일 화면 크기에서는 사용성을 고려하여 화살표 버튼을 중앙 양쪽으로 배치하였습니다.

Instagram

브랜드의 인스타그램 게시글 이미지을 흐르는 배너로 보여주어 메뉴와 sns 계정 홍보 효과를 주었습니다. 이미지 호버 시 브랜드의 인스타그램 계정 아이디가 나타나고, 클릭 시 인스타그램 계정이 새 탭으로 열립니다.
